## Tech Stack

- Vite
- React
- JavaScript
- HTML
- CSS
- SASS

---

## Functionalities

- Rejestracja i logowanie użytkownika
- Tablica Kanban z drag & drop
- Widok szczegółów taska
- Zakładka Profil
- Częściowo zmockowane menu i projekty
- Responsywny layout

## Check it out

```bash
# 1. Sklonuj repozytorium
git clone <LINK_DO_REPO>

# 2. Przejdź do folderu projektu
cd taskify

# 3. Zainstaluj zależności
npm install

# 4. Uruchom aplikację
npm run dev
